# Top-Car-Sales-Analysis-of-10-Car-Dealerships
A Dashboard that analyses Car Sales data of 10 Car Dealerships from South Africa and Botswana in the year 2022. The data used in the Dashboard is composed of 3 datasets all relative to the Car sales of each Dealership. The data was acquired on Kaggle and can be found on this link: https://www.kaggle.com/datasets/teresiawangari/top-cars-south-africa
